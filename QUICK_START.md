# Quick Start - Office Document Conversion

## 🚀 Get Started in 5 Minutes

### Step 1: Get ConvertAPI Secret (2 minutes)
1. Go to https://www.convertapi.com/
2. Click "Sign Up" (free, no credit card)
3. Verify email
4. Copy your API secret from dashboard

### Step 2: Add to Environment (1 minute)
Create or edit `.env.local` file:
```bash
CONVERTAPI_SECRET=secret_xxxxxxxxxxxxx
```

### Step 3: Restart Server (1 minute)
```bash
# Stop server (Ctrl+C)
npm run dev
```

### Step 4: Test (1 minute)
1. Upload a Word/Excel/PowerPoint file
2. Click "View"
3. ✅ Should convert to PDF and open

## That's It!

Your Office documents will now convert properly to PDF with full content.
